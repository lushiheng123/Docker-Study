## 1-写yml文件
![alt text](README_Images/4-拉取镜像指令/image-5.png)
```yml
apiVersion: v1
kind: Pod
metadata:
  name: myapp-pod
  labels:
    app: myapp
    type: back-end
spec:
  containers:
   - name: backend1
     image: express-backend1:latest
```
## 2-`kubectl create -f pod-backend-1.yml`创建pod
